      <p>Inline CSS is written directly inside an HTML tag using the style attribute. Only that particular element is affected. It is quick for small changes but not recommended for large websites.</p>
      <p>Example: <code>&lt;p style="color: pink; text-align: center;"&gt;Hello World&lt;/p&gt;</code></p>
    </div>

    <div class="section-container">
      <h4>2. Internal CSS (Embedded)</h4>
      <p>Internal CSS is written inside a <code>&lt;style&gt;</code> tag within the <code>&lt;head&gt;</code> section of the HTML. It applies to the whole page and is easier to manage than inline CSS.</p>
      <p>Example: <code>&lt;style&gt; p { color: blue; } &lt;/style&gt;</code></p>
    </div>

    <div class="section-container">
      <h4>3. External CSS</h4>
      <p>External CSS is written in a separate .css file and linked to the HTML using a <code>&lt;link&gt;</code> tag. This is the most efficient method for multi-page websites.</p>
      <p>Example: <code>&lt;link rel="stylesheet" href="style.css"&gt;</code></p>
    </div>

      <h4>Benefits of Each Type</h4>
      <ul>
        <li><strong>Inline:</strong> Fast and element-specific.</li>
        <li><strong>Internal:</strong> Easy to style a single page.</li>
        <li><strong>External:</strong> Ideal for multi-page consistency and reusability.</li>

&lt;html&gt;

&lt;head&gt;
&lt;title&gt;JavaScript&lt;/title&gt;

document.write ("My comment");

      errors occur.
    </p>

      <li>Case Sensitive</li>
      <li>White Space</li>

  <div class="section-container">
    <h4>1. Case Sensitive</h4>
    <p>
      You must be careful in naming your variables.
      The variable <strong>myHome</strong> is not equal to <strong>MyHome</strong>.
    </p>

    <p>
      It is important to keep track of your variables so there
      won’t be problems in calling variables, objects or functions.
    </p>

    <p>
      Unlike in HTML, uppercase letters are not equivalent to
      lowercase letters in JavaScript.
    </p>
  </div>

  <div class="section-container">
    <h4>2. White Space</h4>
    <p>
      JavaScript ignores white space as long as it is within the line
      of the code. It is a good practice to use white spaces between
      expressions to make your codes easier to read.
    </p>

<pre>
var name = "Grace"

var name="Grace"
</pre>

    <p>These two variable declarations are accepted.</p>
  </div>

  <div class="section-container">
    <h4>3. Breaking up of codes</h4>
    <p>
      You can break a code line by using a backslash ('\').
      However, do not break the code in between commands
      or else it will not work properly.
    </p>

<pre>
document.write("wow\Philippines");  // Accepted

document.\write("wow Philippines")  // Not accepted
</pre>

    <h4>Characteristics of JavaScript</h4>

    <p><strong>1. Interpreted Language</strong></p>
    <p>
      It is an interpreted language due to the fact that it requires a browser to run.
      JS was created to work together with HTML and requires a web browser in order
      to run. JS source code can easily be seen when you view the source code of a
      JS-enabled webpage.
    </p>

    <p><strong>2. Scripting Language</strong></p>
    <p>
      Scripting languages are treated like programming languages but they are normally
      created to perform simple repetitive tasks. They do not come with their own
      interfaces but they provide instructions to other programs on what to do.
      JS does not contain the program on how the whole webpage looks like, we can
      use it to change certain elements of the webpage through the browser.
    </p>

    <p><strong>3. Client-Side Scripting</strong></p>
    <p>
      There are basically two kinds of scripting for the web – client-side and
      server-side. The key difference between these scripts is the location to
      which they are executed. A server-side script is executed on the actual
      server machine that hosts the webpage while a client-side script is
      executed on the very machine that you are using.
    </p>

    <p><strong>4. Object-Oriented Programming</strong></p>
    <p>
      Another advantage of JS is that it presents many things as objects and lets
      you easily manipulate them. OOP is a programming style that treats every
      element in a system as an object. In OOP, each object has its associated
      functions called methods which are the things that objects know how to do.
    </p>
  </div>


  <div class="section-container">
    <h4>The Key in Understanding JavaScript</h4>

    <p>1. Learn at an optimal pace and do a lot of practice.</p>

    <p><strong>2. Understand the concepts of objects.</strong></p>
    <p>
      Objects – are items that exist in the browser. The browser window,
      the page itself, the status bar, and the date and time stored in the
      browser are all objects.
    </p>

    <p><strong>Methods</strong></p>
    <p>
      Methods are actions to be performed on or by an object.
      Methods also represent functions that are designed for
      specific purposes like doing math or parsing strings.
    </p>

    <p><strong>Properties</strong></p>
    <p>
      Properties are an existing subsection of an object.
      It is also important to learn these basic concepts in programming.
    </p>

    <p><strong>Event</strong></p>
    <p>
      Something which happens or which is done such as loading a page,
      holding a mouse over a link, or clicking a link. You can actually
      write scripts and set them to execute when a specific event occurs.
    </p>

    <p><strong>Parameter</strong></p>
    <p>
      Data or other objects which describe the characteristics of an object.
    </p>

    <p><strong>Type</strong></p>
    <p>
      Type of variable such as integer, string, floating point or Boolean.
    </p>

    <p><strong>Type Conversion</strong></p>
    <p>
      The act of converting a variable from one type to another.
    </p>

    <p>
      A script is a set of computer instructions or commands that puts together
      or connects existing components to accomplish a new related task.
      JS is a powerful scripting language used to create web functionalities.
      A text editor and a web browser are the tools needed in JS scripting.
      The basic components of JS are objects, methods and properties.
    </p>

<h4>What are Events?</h4>
<p>
Events are actions that are triggered either by mouse click, function calls,
keyboard strokes, loading of images, submitting an HTML form, etc.

without a force that will trigger it.
</p>

<ul>
<li><b>onabort</b> – The loading of an image is interrupted.</li>

<li><b>onchange</b> – The user changes the content of a field.</li>
<li><b>onclick</b> – Mouse click of an object.</li>
<li><b>ondblclick</b> – Double mouse click on an object.</li>
<li><b>onerror</b> – An error occurs when loading a document or image.</li>
<li><b>onfocus</b> – An element gets focus.</li>
<li><b>onkeydown</b> – A keyboard key is pressed.</li>
<li><b>onkeypress</b> – A keyboard key is held down.</li>
<li><b>onkeyup</b> – A keyboard key is released.</li>
<li><b>onload</b> – A page or image is finished loading.</li>
<li><b>onmousedown</b> – A mouse button is pressed.</li>
<li><b>onmousemove</b> – A mouse is moved.</li>
<li><b>onmouseout</b> – The mouse is moved off an element.</li>
<li><b>onmouseover</b> – The mouse is moved over an element.</li>
<li><b>onmouseup</b> – A mouse button is released.</li>
<li><b>onreset</b> – The reset button is clicked.</li>
<li><b>onresize</b> – A window or frame is resized.</li>
<li><b>onselect</b> – Text is selected.</li>

A cookie is a message given to a web browser.
</p>

    <p>Kapag tumataas ang presyo, tataas ang dami ng gusto at kayang ibenta, at kapag bumababa ang presyo, bababa naman ang dami ng gusto at kayang ibenta. (Ceteris Paribus)</p>
  </div>

  <div class="section-container">
    <h4>Mathematical Formula</h4>
    <p><strong>Supply = Pagnanais + Kakayahan</strong></p>
  </div>

  <div class="section-container">
    <h4>Supply Schedule</h4>
    <p>Talahanayang nagpapakita kung gaano karaming produkto o serbisyo ang nais gawin ng mga prodyuser.</p>
  </div>

  <div class="section-container">
    <h4>Supply Curve</h4>
    <p>Grapikong representasyon kung gaano karaming produkto o serbisyo ang nais gawin ng mga prodyuser.</p>
  </div>

  <div class="section-container">
    <h4>Supply Function</h4>
    <p>Isang mathematical equation na nagpapakita sa relasyon ng dami:</p>
    <p><strong>Qs = c + bP</strong></p>
    <ul>
      <li>Qs = dami ng supply</li>
      <li>c = dami ng supply kung ang presyo ay 0</li>
      <li>P = presyo</li>
      <li>b = slope ng supply function - interval na pagbabago</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>Mga Salik ng Supply</h4>
    <ul>
      <li>Pagbabago sa teknolohiya</li>
      <li>Pagbabago sa halaga ng mga salik ng produksyon</li>
      <li>Pagbabago sa bilang ng mga nagtitinda</li>
      <li>Pagbabago sa presyo ng kaugnay na produkto (Complementary at Substitute Products)</li>
      <li>Ekspektasyon ng presyo</li>
      <li>Okasyon/Tradisyon</li>
      <li>Panahon/Klima</li>
      <li>Dali ng pagkasira / pagkabulok ng isang produkto</li>
      <li>Kalamidad</li>
      <li>Subsidiya / tulong mula sa pamahalaan</li>
    </ul>

  <h3 class="lesson-title">Lesson 3: Price Elasticity</h3>
  
  <div class="section-container">
    <h4>Elastisidad</h4>
    <p>Pamamaraan ng pagsukat ng tugon ng mga mamimili at nagtitinda sa pagbabago ng presyo.</p>
  </div>

  <div class="section-container">
    <h4>1. Elastisidad ng Demand</h4>
    <p>Pagsukat ng porsyento ng pagtugon ng mamimili sa pagbabago ng presyo.</p>
  </div>

  <div class="section-container">
    <h4>2. Elastisidad ng Supply</h4>
    <p>Pagsukat ng porsyento ng pagtugon ng prodyuser sa pagbabago ng presyo.</p>
  </div>

  <div class="section-container">
    <h4>Mga Uri ng Elastisidad</h4><br>
    <ul>
      <li><strong>Elastic:</strong> Ang demand o supply ay matatawag na price elasticity kapag mas malaki ang bahagda ng pagtugon ng quantity demanded/supplied kaysa sa bahagda ng pagbabago ng presyo. /E &gt; 1/</li><br>
      <li><strong>Di-Elastic:</strong> Maliit ang naging bahagda ng pagtugon ng demand/supply kaysa sa bahagda ng pagbabago ng presyo. /E &lt; 1/</li><br>
      <li><strong>Unitary:</strong> Pareho ang bahagda ng pagbabago ng presyo sa bahagda ng pagbabago ng quantity demanded. /E = 1/</li><br>
      <li><strong>Ganap na Elastik:</strong> Ang pagbabago ng presyo ay nagdudulot ng infinite na pagbabago sa quantity demanded/supplied. /E = &infin;/</li><br>
      <li><strong>Di-Ganap na Elastik:</strong> Hindi tumutugon sa pagbabago ng presyo; kahit ang presyo ay bibilhin pa rin ang kaparehong dami ng produkto. /E = 0/</li>
    </ul>
  </div>

  <h3 class="lesson-title">Lesson 4: Market Equilibrium</h3>
  
  <div class="section-container">
    <h4>Market Equilibrium</h4>
    <p>
      Ito ay isang kalagayan na walang sinumang mamimili at nagbibili ang gustong baguhin ang kasalukuyang sitwasyon sa pamilihan. 
      Ipinapakita ng ekwilibriyo ang pagkakasundo ng bumubili at nagbibili sa isang takdang presyo at dami ng produkto.
    </p>
  </div>

  <div class="section-container">
    <h4>Talahanayan</h4>
    <p>Maaari ding makuha ang equilibrium price sa pamamagitan ng demand at supply function.</p>
  </div>

  <div class="section-container">
    <h4>Tandaan</h4>
    <ul>
      <li>Kung bumaba ang presyo, gaganahan ang mamimili na bumili at magkakaroon ng shortage dahil sa kakulangan ng supply.</li><br>
      <li>Kapag alinman sa isa ang tumaas o bumaba at hindi na ito balance, magkakaroon ng disequilibrium.</li><br>
      <li>Kung tumaas ang presyo, ang mga prodyuser ay gaganahang magprodyus ng marami, ngunit ang pagtaas ng presyo ay makakabawas sa kakayahan ng mamimili na bumili. Magkakaroon ng labis na supply o surplus.</li>
    </ul>

<div class="lesson">
  <h3 class="lesson-title">Lesson 5: Iba’t Ibang Estruktura ng Pamilihan</h3>
  
  <div class="section-container">
    <h4>Pamilihan</h4>
    <ul>
      <li>Ang pamilihan ay ang nagsisilbing tagpuan ng mga prodyuser at konsyumers.</li>
      <li>Ayon kay Adam Smith, may "invisible na kamay" (presyo) ang gumagabay sa prodyuser at konsyumer upang maisaayos ang pamilihan.</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>Istraktura ng Pamilihan</h4>
    <p>Tumutukoy sa balangkas na umiiral sa sistema ng pamilihan kung saan ipinapakita ang ugnayan ng konsyumer at prodyuser.</p>
  </div>

  <div class="section-container">
    <h4>Ganap na Kompetisyon</h4>
    <ul>
      <li>Maraming maliliit na konsyumer at prodyuser.</li>
      <li>Magkakatulad ang produkto.</li>
      <li>Malayáng paggalaw ng sangkap ng produksyon.</li>
      <li>Malayáng nakakapasok o nakakalabas sa industriya.</li>
      <li>Malayáng impormasyon ukol sa pamilihan.</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>Di-Ganap na Kompetisyon</h4>
    <p>Sa istrakturang ito ay may kakayahan na maimpluwensyahan ang presyo sa pamilihan.</p>
  </div>

  <div class="section-container">
    <h4>Monopolyo</h4>
    <ul>
      <li>Iisa lamang ang nagtitinda o kumokontrol.</li>
      <li>Walang direktang kapalit.</li>
      <li>Ang produkto ay lubhang mahalaga.</li>
      <li>May kakayahan hadlangan ang mga kalaban.</li>
      <li>Ang mga konsyumer ay napipilitang tanggapin ang pagiging makapangyarihan ng monopolista.</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>Monopsonyo</h4>
    <p>Mayroon lamang iisang konsyumer ngunit maraming prodyuser ng produkto o serbisyo.</p>
  </div>

  <div class="section-container">
    <h4>Oligapolyo</h4>
    <p>Kaunti lamang ang prodyuser, ngunit marami ang konsyumer.</p>
  </div>

  <div class="section-container">
    <h4>Monopolistikong Kompetisyon</h4>
    <ul>
      <li>Marami ang prodyuser at konsyumer.</li>
      <li>May kakayahan ang prodyuser na magtakda ng sarili niyang presyo.</li>
      <li>Product Differentiation.</li>
    </ul>
  </div>

  <h3 class="lesson-title">Lesson 6: Price Control</h3>
  
  <div class="section-container">
    <h4>Bahaging Ginampanan ng Pamahalaan sa Regulasyon ng mga Gawaing Pangkabuhayan</h4>
    <p>Paraan ng pagkontrol sa presyo:</p>
    <ul>
      <li>Price Ceiling</li>
      <li>Price Floor</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>Price Control</h4>
    <p>Ang price control ay ang ipinatutupad ng gobyerno sa presyo ng mga bilihin at serbisyo upang mapanatili ang kanilang abot-kayang halaga o upang masiguro ang minimum na kita ng mga prodyuser.</p>
  </div>

  <div class="section-container">
    <h4>Republic Act 7581</h4>
    <p>Kilala bilang Price Control Act. Layunin nito na maisagawa ng pamahalaan ang pagkontrol sa presyo ng mga bilihin.</p>
  </div>

  <div class="section-container">
    <h4>National Price Coordination Council</h4>
    <p>Layunin nitong mabantayan ang presyo ng mga produkto pagkatapos magpalabas ng price ceiling ang pamahalaan.</p>
  </div>

  <div class="section-container">
    <h4>Price Ceiling</h4>
    <ul>
      <li>Pinakamataas na presyong itinakda ng pamahalaan upang ipagbili ang mga produkto.</li>
      <li>Isinasagawa upang protektahan ang mga mamimili laban sa abusadong negosyante.</li>
      <li>Itinakda mas mababa sa equilibrium price na umiiral sa pamilihan.</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>Price Freeze</h4>
    <p>Pagbabawal sa pagtataas ng presyo sa pamilihan. Ipinatutupad upang mapigilan ang pananamantala ng mga negosyante.</p>
  </div>

  <div class="section-container">
    <h4>Floor Price</h4>
    <p>Pinakamababa na presyong itinakda ng pamahalaan upang ipagbili ang mga produkto.</p>
  </div>

  <div class="section-container">
    <h4>Advantage ng Price Control</h4>
    <ul>
      <li>Napananatili ang abot-kayang presyo para sa mga mamimili.</li>
      <li>Naipagtatanggol ang kapakanan ng mahihirap.</li>
      <li>Napananatili ang katatagan ng pamilihan.</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>Disadvantage ng Price Control</h4>
    <ul>
      <li>Magkakaroon ng kakulangan ng supply.</li>
      <li>Mawawalan ng gana ang prodyuser na magtinda.</li>
      <li>Magkakaroon ng illegal na bentahan o black market.</li>
    </ul>
  </div>

  <h3 class="lesson-title">Lesson 3: Industriya</h3>

  <div class="section-container">
    <h4>Konsepto</h4>
    <p>Tumutukoy sa pagproseso ng mga hilaw na materyal at paggawa ng mga produkto na kinakailangan para sa pampublikong pamilihan
     Naglalayong maiproseso ang mga hilaw na materyal upang makabuo ng produkto na ginagamit ng tao
    </p>
  </div>

  <div class="section-container">
    <h4>Sub-Sektor</h4>
    <ul>
      <li>Pagmimina – Pagkuha at pagproseso ng mga yamang mineral (metal, di-metal, o enerhiya) upang gawing tapos na produkto o kabahagi ng isang yaring kalakal.
       Ang mismong produkto, hilaw man o naproseso ay nagbibigay ng kita para sa bansa
      </li>
      <li>Pagmamanupaktura – Paggawa ng mga produkto sa pamamagitan ng manual labor o ng mga makina.
       Nagkakaroon ng pisikal o  kemikal na transpormasyon ang mga materyal o bahagi nito sa pagbuo ng mga bagong produkto
      </li>
      <li>Konstruksyon – Pagtatayo ng mga gusali, estruktura, at iba pang land improvements</li>
      <li>Utilities – Pagtugon ng mga pangunahing pangangailangan ng mga mamamayan tulad ng tubig, kuryente, at gas
       Paglalatag ng mga imprastraktura at angkop na teknolohiya upang maihatid ang serbisyo</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>KAHALAGAHAN NG SEKTOR NG INDUSTRIYA SA PAMBANSANG KAUNLARAN</h4>
    <ol>
      <li>Tumatanggap ng mga produkto mula sa sektor ng agrikultura at ng serbisyo mula sa sektor ng paglilingkod</li>
      <li>Nagkakaloob ng hanapbuhay sa maraming Pilipino</li>
      <li>Nagpapasok ng dolyar sa bansa</li>
      <li>Nakagagamit ng makabagong Teknolohiya</li>
      <li>Gumagawa ng mga intermediate products (mga materyales o produkto na ginagamit sa paggawa ng iba pang produkto; hal. wheat - to make bread)</li>
    </ol>
  </div>

  <div class="section-container">
    <h4>Pagkaka-ugnay ng Agrikultura at Industriya</h4>
    <ul>
     <li>Nagmumula sa agrikultura ang mga hilaw na sangkap na ginagamit sa pagbuo ng mga produkto ng industriya</li>
     <li>Ang mga ginagamit na traktora,sasakyang pangingisda, at iba pangprodukto ay mula sa industriya naginagamit ng mga magsasaka upangmagkaroon ng mataas na produksyonna magbibigay ng malaking kita sa sektor ng agrikultura.</li>
     <li>Ang dami ng lakas-paggawa sa mga mamimili ay nagpapakita ng lubos na pagtutulungan sa mga sector ng industriya at agrikultura.</li>
     <li>Ang agrikultura ang pinagmumulan ng pagkain ng mga industriya na siyang tumutugon sa mga pagkaing kinakailangan ng mga tao.</li>
     <li>Ang industriya ang nagbibigay ng malaking kontribusyon sa ekonomiya.</li>
     <li>Ang mga sangkap ay nagkakaroon ng transpormasyon dahil sa mga lakas–paggawa na gumagawa upang madagdagan ang halaga nito at lalo pang gumanda.</li>
  </div>

  <div class="section-container">
    <h4>Patakaran sa Ekonomiya</h4>
    <ul>
      <li>Filipino First Policy : Nagsimula sa programang FIlipino Muna ni Pangulong Carlos P. Garcia. Naglalayong bigyan ng higit na malaking partisipasyon ang mga Pilipino sa pagpapatakbo ng ekonomiya</li>
      <li>Oil Deregulation Law (R.A. 8479) : Nagbukas ng kompetisyon sa industriya ng langis. Ang pamilihan ang nagdidikta sa presyo ng petrolyo, at hindi ito maaaring pakialaman ng gobyerno</li>
      <li>Microfinancing : Serbisyong pinansiyal na isinasagawa ng mga bangko na magpautang na mayroong mababang interes sa mahihirap na pamilya</li>
      <li> Online Business </li>
      <li> Retail Nationalization Act (R.A. 1180) : Sampung taong palugit sa mga dayuhang capitalist. Pilipino na may korporasyon o asosasyon lamang ang nasa kalakalang pagtitingi </li>
      <li>Retail Trade Liberalization Act (R.A. 8762) : Nagpawalang-bisa sa RA 1180. Binuksan ang kalakalang pagtitingi sa mga dayuhan</li>
      <li>Build-Operate-Transfer (R.A. 6957) : Binibigyan ng karapatang bumuo ng mga pasilidad sa kanilang mga sariling pondo at palakarin ito sa humigit-kumulang na 25 taon</li>
      <li>Asset Privatization Trust (APT) : Itinatag upang mapangasiwaan ang pagbebenta ng mga korporasyong pagmamay-ari ng pamahalaan sa pribadong pagmamay-ari. Pinasimula ni Corazon Aquino</li>
    </ul>

  <h3 class="lesson-title">Lesson 5: Impormal na Sektor</h3>

  <div class="section-container">
    <h4>Impormal na Sektor</h4>
    <ul>
      <li>Sektor ng ekonomiya na walang pormal o salat sa mga dokumentong kailangan sa pagsasagawa ng gawaing pang-ekonomiya.</li>
      <li>Ang kita ng sektor na ito ay HINDI naisasama sa kabuuang Gross Domestic Product (GDP) ng bansa.</li>
      <li>Paraan ng mamamayan upang magkaroon ng kabuhayan/dagdag kita sa tuwing panahon ng pangangailangan.</li>
      <li>Ito ay nagdudulot rin ng pagkakalikha ng mga produkto at serbisyong tutugon sa ating mga pangangailangan.</li>
      <li>Kilala rin sa tawag na Underground Economy.</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>Halimbawa ng mga Taong Kabilang sa Sektor na Ito</h4>
    <ol>
      <li>Nagtitinda sa kalsada</li>
      <li>Pedicab driver</li>
      <li>Karpintero</li>
      <li>Hindi rehistradong pampublikong sasakyan</li>
    </ol>
  </div>

  <div class="section-container">
    <h4>Katangian ng Impormal na Sektor</h4>
    <ol>
      <li>Hindi nakarehistro sa pamahalaan.</li>
      <li>Hindi nagbabayad ng buwis mula sa kinita.</li>
      <li>Walang legal at pormal na balangkas mula sa pamahalaan para sa pagnenegosyo.</li>
    </ol>
  </div>

  <div class="section-container">
    <h4>Iba’t Ibang Anyo ng Impormal na Sektor</h4>
    <ol>
      <li>Mga gawaing maaaring isakatuparan lamang sa loob ng tahanan.</li>
      <li>Mga gawaing pansibiko, pagkakawanggawa, at panrelihiyon.</li>
      <li>Mga sari-sari stores, pagtitinda sa labas ng bahay o sidewalk, paglalako ng kalakal o serbisyo.</li>
      <li>Ilegal na gawain tulad ng pagnanakaw, pagbebenta ng ilegal na gamot, piracy, at prostitusyon.</li>
    </ol>
  </div>

  <div class="section-container">
    <h4>Kahalagahan ng Impormal na Sektor</h4>
    <ol>
      <li>Sinasalo nito ang mga mamamayan na hindi makapasok bilang mga regular na empleyado sa isang kompanya.</li>
      <li>Nagbibigay ng pagkakataon sa maraming Pilipino na makapaghanapbuhay.</li>
      <li>Nagsisilbing tagasalo ng mga mamamayang may mahigpit na pangangailangan.</li>
      <li>Malaki ang naitutulong sa mga konsyumer ang mga kalakal at serbisyong mula sa impormal na sektor dahil sa MURA nitong halaga.</li>
    </ol>
  </div>

  <div class="section-container">
    <h4>Dahilan Kung Bakit Pumapasok sa Impormal na Sektor ang mga Mamamayan</h4>
    <ol>
      <li>Makaligtas sa pagbabayad ng buwis sa pamahalaan.</li>
      <li>Makaiwas sa masyadong mahaba at masalimuot na proseso ng pakikipagtransaksyon sa pamahalaan (bureaucratic red tape).</li>
      <li>Kawalan ng regulasyon mula sa pamahalaan na kung saan ang mga batas at programa ay hindi naitutupad nang maayos.</li>
      <li>Makapaghanapbuhay nang hindi nangangailangan ng malaking kapital o puhunan.</li>
      <li>Malabanan ang matinding kahirapan.</li>
    </ol>
  </div>

  <div class="section-container">

  <h3 class="lesson-title">Lesson 6: Kalakalang Panlabas</h3>

  <div class="section-container">
    <h4>KALAKALANG PANLABAS</h4>
    <ul>
      <li>Palitan ng produkto at serbisyo sa pagitan ng mga bansa</li>
      <li>Ito ay nagaganap sapagkat walang bansa ang maaaring makatugon sa lahat ng kanyang pangangailangan ng walang tulong mula sa ibang bansa.</li>
      <li>Bilateral: 2 bansa (hal. U.S at PH)</li>
      <li>Bloc: Organisasyon at 1 bansa (hal. APEC at PH)</li>
      <li><strong>EKSPORT (Pagluluwas):</strong> pagpapadala ng mga produkto at serbisyo sa ibang bansa</li>
      <li><strong>IMPORT (Pag-aangkat):</strong> pagbili ng kalakal mula sa ibang bansa</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>DAHILAN NG KALAKALANG PANLABAS</h4>
    <ol>
      <li>Pagkakaiba ng Teknolohiya</li>
      <li>Pagkakaiba sa Pinagkukunang Yaman</li>
      <li>Pagkakaiba sa Panlasa</li>
      <li>Pagkakaiba sa Halaga ng Produksyon</li>
    </ol>
  </div>

  <div class="section-container">
    <h4>PATAKARANG UMAAPEKTO SA KALAKALANG PANLABAS</h4>

    <p><strong>A. Taripa / Tariff:</strong></p>
    <ul>
      <li>Buwis sa mga produktong inaangkat</li>
      <li>Nagpapataas ng presyo at nagpapababa ng benta ng inaangkat ng produkto</li>
      <li>Nagpapataw ng mataas na taripa pang mapigil ang pagpasok ng dayuhang produktong kalaban ng lokal na produkto.</li>
    </ul>

    <p><strong>B. Kota (Quota):</strong></p>
    <ul>
      <li>Maaaring limitahan (kotahan) ang pagpasok ng mga kalabang produkto</li>
    </ul>

    <p><strong>C. Subsidy:</strong></p>
    <ul>
      <li>Tulong na ibinibigay ng gobyerno upang bumaba ang halaga ng produksyon ng mga lokal na produkto.</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>BATAYAN NG KALAKALANG PANLABAS (David Ricardo)</h4>

    <p><strong>Absolute Advantage</strong></p>
    <ul>
      <li>Nakakalikha ng mas maraming bilang ng produkto gamit ang mas kaunting salik ng produksyon kumpara sa ibang bansa.</li>
    </ul>

    <p><strong>Comparative Advantage</strong></p>
    <ul>
      <li>Ang bansa ay magkakaroon ng espesyalisasyon sa paglikha ng kalakal.</li>
      <li>Kapag kaya ng bansa gawin ang kalakal ng mas efficient kumpara sa ibang bansa.</li>
      <li>Hal. Cebu may comparative advantage sa pagluto ng Lechon</li>
    </ul>

    <p><strong>Balance of Trade (BOT)</strong></p>
    <ul>
      <li>Kalagayan ng kabayaran ng pagluluwas (export) at sa pag-aangkat (import).</li>
      <li>Trade Deficit: import > export</li>
      <li>Trade Surplus: import < export</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>KABUTIHAN NG PAKIKIPAGKALAKALAN</h4>
    <ul>
      <li>Mas maraming produkto ang mapagpipilian sa pamilihan.</li>
      <li>Napapataas nito ang kalidad ng mga produkto na nabibili sa pamilihan</li>
      <li>Nagkakaroon ng pagkakaunawaan ang mamamayan ng bansang nakikipagkalakalan.</li>
      <li>Lumalawak ang pamilihan ng mga produkto sa bansa.</li>
    </ul>
  </div>

  <div class="section-container">
    <h4>DI-KABUTIHAN NG PAKIKIPAGKALAKALAN</h4>
    <ul>
      <li>Nagiging palaasa ang mga mamamayan sa produktong imported</li>
      <li>Hindi nalilinang ang pagkamalikhain ng mga mamamayan dahil umaasa na lamang sila sa mga produktong gawa sa ibang bansa</li>
      <li>Pagkawala ng sariling pagkakakilanlan bunga ng pagpasok ng dayuhang kultura sa lipunan</li>
    </ul>
  </div>

  <div class="section-container">

    <p><strong>World Trade Organization (WTO)</strong></p>
    <ul>
      <li>Pinasinayaan at nabuo noong Enero 1, 1995</li>
      <li>Samahang namamahala sa pandaigdigang patakaran ng sistema ng kalakalan o global trading system sa pagitan ng mga kasaping estado o member states</li>

    <p><strong>Layunin</strong></p>
    <ul>
      <li>Pagsusulong ng maayos na kalakalan sa pamamagitan ng pagpapababa ng hadlang sa kalakalan (trade barriers)</li>
      <li>Plataporma para sa negosasyon at tulong-teknikal sa developing countries.</li>
      <li>Paggamit ng trade sanction sa mga kasaping hindi sumusunod sa desisyon ng samahan</li>
    </ul>

    <p><strong>Asia-Pacific Economic Council (APEC)</strong></p>

    <p><strong>Layunin</strong></p>
    <ul>

      <li>Pagpapabilis at pagpapadali ng pagnenegosyo</li>
      <li>Pagtutulungang pang-ekonomiya at teknikal</li>
    </ul>

    <p><strong>Association of Southeast Asian Nations (ASEAN)</strong></p>
    <ul>
      <li>Paunlarin at isulong ang malayang kalakalan sa bawat kasapi ng ASEAN pati ang mga bansang dialogue partner nito.</li>
      <li>Itinatag noong August 8, 1967</li>
    </ul>

    <p><strong>Layunin</strong></p>
    <ul>
      <li>Upang maging ganap ito, nabuo ang ASEAN Free Trade Area (AFTA) noong January 28, 1992, sa Singapore.</li>
      <li>Nakabatay sa Common Effective Preferential Tariff (CEPT) na nag-aalis ng taripa at quota para sa mas malayang daloy ng produkto sa ASEAN.</li>
